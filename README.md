# SecureMessageEncryptionandDecryption
Overview

This is a Python-based GUI application that allows users to securely encrypt and decrypt messages using a password. The application features a modern, user-friendly interface built with Tkinter and incorporates Base64 encoding for encryption.

Features

Encrypt Messages: Securely encode text messages using Base64.

Decrypt Messages: Decode encrypted messages with a password.

Password Protection: Ensures that only authorized users can access encrypted data.

File Handling: Open text files for encryption/decryption and save encrypted messages.

Clipboard Utility: Copy encrypted or decrypted messages to the clipboard.

Modern UI: Styled with Tkinter's ttk module for an enhanced user experience.

Requirements

Make sure you have the following dependencies installed before running the application:

pip install pyperclip

Installation

Clone the repository:

git clone https://github.com/yourusername/secure-message-encryption.git
cd secure-message-encryption

Run the Python script:

python encryption_app.py

Usage

Open the application.

Enter the message you want to encrypt.

Provide the password for encryption.

Click the Encrypt button to secure the message.

Copy or save the encrypted text.

To decrypt, enter the encrypted text, provide the correct password, and click Decrypt.

File Handling

Open File: Load a text file for encryption or decryption.

Save File: Save encrypted messages as a .txt file.
